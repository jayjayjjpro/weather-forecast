## Overview

The **Weather Tracking App** allows users to track current weather conditions of various cities worldwide. It provides an intuitive user interface that fetches live weather data â€” such as temperature, humidity, and weather conditions â€” using a third-party weather API. Weather information is displayed on an interactive global map, focusing on predefined or popular cities.

The app is built using **Node.js** for the backend, **React.js** with **Vite** for the frontend, and **PostgreSQL** (via Neon) for storing data about the most popular tracked cities. The app is cloud-hosted and supports automated deployments via **GitHub Actions**.
